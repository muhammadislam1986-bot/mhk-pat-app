MHK PAT Mobile V2

How to use:
1. Upload the folder or zip contents to any static web host.
2. Open the hosted URL in Chrome on Android.
3. Use Chrome menu > Install app / Add to home screen.

Easy hosting options:
- Netlify Drop
- Cloudflare Pages
- GitHub Pages

Notes:
- The app saves data in the browser on the device.
- Use JSON backup regularly.
- Engineer signature is included on the report.
